Notas da versão

Esta página documenta as atualizações de produção do Memorystore para Memcached. Acesse-a periodicamente para ver avisos de recursos novos ou atualizados, correções de bugs, problemas conhecidos e recursos obsoletos.

Você pode ver as atualizações mais recentes de todos os produtos do Google Cloud na página Notas de versão do Google Cloud .

Para receber as atualizações de produtos mais recentes, adicione o URL desta página ao leitor de feeds ou adicione o URL do feed diretamente: https://cloud.google.com/feeds/memorystorememcache-release-notes.xml
