リリースノート

このページには、Cloud Source Repositories の本番環境に関する更新情報が記載されています。このページを定期的にチェックして、新機能や更新された機能、バグ修正、既知の問題、非推奨となった機能に関するお知らせを確認してください。

Google Cloud リリースノート のページで、Google Cloud の最新のプロダクト更新情報をすべて確認できます。

プロダクトのアップデートに関する最新情報を受け取るには、このページの URL を フィード リーダー に追加するか、またはフィード URL ディレクトリ https://cloud.google.com/feeds/cloud-source-repositories-release-notes.xml を直接追加します。
