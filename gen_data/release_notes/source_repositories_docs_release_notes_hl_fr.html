Notes de version

Cette page répertorie les mises à jour en production de Cloud Source Repositories. Consultez-la régulièrement pour obtenir des informations sur les fonctionnalités nouvelles ou mises à jour, les corrections de bugs, les problèmes connus et les fonctionnalités obsolètes.

Vous pouvez consulter les dernières mises à jour de produits pour l'ensemble de Google Cloud sur la page Notes de version de Google Cloud .

Pour recevoir les dernières mises à jour concernant ce produit, ajoutez l'URL de cette page à votre lecteur de flux ou ajoutez l'URL du flux directement : https://cloud.google.com/feeds/cloud-source-repositories-release-notes.xml

  • Mettre en miroir un dépôt Bitbucket
