September 05, 2017

CHANGED:

Version 1.0.0.9

FEATURE:

Added the New-GceNetwork and Remove-GceNetwork cmdlets to add and remove Compute Engine Networks.

FEATURE:

Added the -MaxBadRecords parameter to the Start-BqJob cmdlet .

FIXED:

Fixed a bug in the Fix-GkeCluster cmdlet where the -NodeVersion parameter was not processed correctly.
