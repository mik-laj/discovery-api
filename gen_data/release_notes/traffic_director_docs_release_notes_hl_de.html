Für neueste Produktaktualisierungen können Sie die URL der Seite in Ihren Feedreader einfügen oder die Feed-URL direkt hinzufügen: https://cloud.google.com/feeds/trafficdirector-release-notes.xml .

July 21, 2020

FEATURE:

Traffic Director supports proxyless gRPC applications in General Availability. In this deployment model, gRPC applications can participate in a service mesh without needing a sidecar proxy.

FEATURE:

FEATURE:
