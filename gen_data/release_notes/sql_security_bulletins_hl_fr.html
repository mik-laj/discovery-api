Bulletins de sécurité

Cette page répertorie les bulletins de sécurité pour Cloud SQL.

Pour recevoir les derniers bulletins de sécurité, ajoutez l'URL de cette page à votre lecteur de flux . Vous pouvez également ajouter l'URL du flux directement : https://cloud.google.com/feeds/cloud-sql-security-bulletins.xml

5 juin 2020

Description Niveau de gravité Notes

La version mineure de MySQL 5.6 est mise à niveau vers la version 5.6.42. La version mineure de MySQL 5.7 est mise à niveau vers la version 5.7.25.

Impact sur Cloud SQL

Les instances Cloud SQL pour MySQL ont été automatiquement mises à jour dans le cadre de la mise à jour de maintenance.

Que dois-je faire ?

Aucune action spéciale n'est requise de votre part.

La version mineure de PostgreSQL 9.6 est mise à niveau vers la version 9.6.16. La version mineure de PostgreSQL 10 est mise à niveau vers la version 10.11. La version mineure de PostgreSQL 11 est mise à niveau vers la version 11.6. La version mineure de PostgreSQL 12 est mise à niveau vers la version 12.1. Reportez-vous aux informations de sécurité PostgreSQL pour obtenir des informations sur les failles spécifiques traitées dans la mise à jour.

Impact sur Cloud SQL

Les instances Cloud SQL pour PostgreSQL ont été mises à jour automatiquement dans le cadre de la mise à jour de maintenance.

Que dois-je faire ?

Aucune action spéciale n'est requise de votre part.
